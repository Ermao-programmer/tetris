# tetris
俄罗斯方块游戏。
- [x] 游戏基础玩法（随机四格骨牌、骨牌旋转、骨牌移动、消除行、计分）
- [x] 暂停、恢复和重新开始游戏
- [ ] 窗口大小自适应
- [ ] 支持web
